## CRUD

- Create

  - `insertOne(data, options)` — insert a single document
  - `insertMany(dataArray, options)` — insert multiple documents

- Read

  - `find(filter, projection)` — find documents matching a filter
  - `findOne(filter, projection)` — find a single document

- Update

  - `updateOne(filter, update, options)` — update a single matching document
  - `updateMany(filter, update, options)` — update multiple matching documents
  - `replaceOne(filter, replacement, options)` — replace a single document

- Delete

  - `deleteOne(filter, options)` — delete a single matching document
  - `deleteMany(filter, options)` — delete multiple matching documents

Notes:

- These commands mirror the CRUD diagram: Create (C) → insert*, Read (R) → find*, Update (U) → update*/replaceOne, Delete (D) → delete*.
