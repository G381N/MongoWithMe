# 1. Aggregation — Introduction

📹 **Video Reference:** [MongoDB Aggregation Pipeline](https://www.youtube.com/watch?v=A3jvoE0jGdE)

---

## What is Aggregation?

With `find()`, you can filter and get documents. But what if you want to **calculate** things — like "how many users are in each country?" or "what's the average age by gender?" That's where aggregation comes in.

The **Aggregation Framework** lets you process data through a **pipeline** — a series of steps (called **stages**) where each stage transforms the data before passing it to the next one.

Think of it like a factory assembly line:

```
Raw Data → [ Stage 1 ] → [ Stage 2 ] → [ Stage 3 ] → Final Result
```

Each stage does one job. Together, they answer complex questions that `find()` alone can't.

---

## Why Use Aggregation?

- **Grouping** — count users per country, average salary per department
- **Reshaping** — pull nested fields up, rename fields, remove unnecessary data
- **Calculating** — sum, average, min, max across documents
- **Sorting & Limiting** — get the top N results
- **Joining** — combine data from multiple collections

---

## How Stages Work

A pipeline is just an **array of stages**. Each stage is an object with a stage operator:

```
Document → [Stage] → Document → [Stage] → Document → [Stage] → Document
```

The output of one stage becomes the input of the next. The order matters — data flows left to right.

---

## Basic Syntax

```javascript
db.<collection>.aggregate([
  { <stage1> },
  { <stage2> },
  { <stage3> }
])
```

An empty pipeline returns all documents (same as `find()`):

```javascript
db.persons.aggregate([])
```

## Loading the Sample Data

First, download the [`Persons.json`](Persons.json) file from this directory. Then load it into your database under a collection called `persons`:

```bash
mongoimport --db GebinDB --collection persons --file Persons.json --jsonArray
```

Verify it loaded:

```javascript
use GebinDB
```

```javascript
db.persons.aggregate([])
```

You should see all the person documents. Each one looks like this:

